# Nesting in Scss

## What is Advantage of Nesting in scss.
   Here you can able to nested html elements for access child properties for styling.

## How can we enable nesting

> <code>
>    ul{
>       list-style: none;
>       display: flex;
>
>       li{
>        margin-right: 1rem;
>          a{
>                text-decoration: none;
>                font-size: 1.5rem;
>                color: black;
>            }
>        }
>    }
</code>
